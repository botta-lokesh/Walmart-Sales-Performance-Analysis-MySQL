# Walmart Sales Performance Analysis using MySQL

## Project Overview

This project analyzes Walmart sales transaction data using MySQL to uncover business insights, sales trends, customer behavior, and product performance. The analysis demonstrates how SQL can be used to transform raw data into actionable business intelligence for decision-making.

---

## Business Objectives

- Analyze overall sales performance
- Identify top-performing branches
- Evaluate product line profitability
- Understand customer purchasing behavior
- Discover revenue trends and patterns
- Generate actionable business recommendations

---

## Dataset Information

Source: Walmart Sales Dataset

The dataset contains transactional sales records including:

- Invoice ID
- Branch
- City
- Customer Type
- Gender
- Product Line
- Unit Price
- Quantity
- Revenue
- Payment Method
- Customer Rating
- Date & Time

## Key Business Questions Solved

1. Which branch generated the highest revenue?
2. Which product line contributed the most sales?
3. What are the peak sales periods?
4. Which customer segment is most profitable?
5. Which payment methods are most preferred?
6. Which cities contribute the highest revenue?
7. How do customer ratings vary across branches?
8. What are the monthly sales trends?
